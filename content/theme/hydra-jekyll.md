# A simple and minimalist theme & full featured jekyll theme

This theme is a lightweight starter kit. It also gives you a well organised starting point to extend it for yourself.

## Features

- Bootstrap
- Responsive Design
- Dark/Light Theme
- Custom 404 Page
- About Page
- Tags Page
- Code Highlighter
- Social Share
- Atom & Json feeds
- Robots.txt
- Sitemap
- Next & Previous Post
- Pagination
- Disqus
- Mathjax Support
- Google Analytics
