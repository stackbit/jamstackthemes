Designed for a hyper focused blog. Made with the static site generator Eleventy 2.0+ and the CSS framework Bootstrap 5+

* Bootstrap 5 
* Dev Server 
* Minimizing and optimizing HTML and CSS output 
* Automated jpg -> webP image generation 
* Ready for Netlify
* GitHub template
