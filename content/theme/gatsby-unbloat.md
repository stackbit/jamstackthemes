# Gatsby Unbloat starter

Reduce your footprint to the minimum. No option, no React, no extra js, no analytics. The theme removes lots of features you might not need.

## Features

* Easy CSS styles
* humans.txt
* Easy to deploy
* Very small page size
* SEO friendly
