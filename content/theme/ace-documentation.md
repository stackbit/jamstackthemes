# Ace documentation theme for Hugo

Ace documentation offers a feature-complete documentation theme for Hugo, powered by Bootstrap 4. 

## Features

* Syntax highlighting
* Full search
* Copyable code blocks
* Responsive layout
* All Bootstrap 4 components
* SCSS compiler built-in
* Useful shortcodes
* Font Awesome icons
